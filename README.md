## 📖 Course Description

This repository contains **in-class exercises** developed during the lectures of the **Advanced Algorithms** course offered at PUCP during the 2026-1 academic semester.

The course is of a theoretical-practical nature, whose purpose is for the student to understand the implementation and application of algorithms of greater complexity. Topics developed include backtracking algorithms, dynamic programming, genetic algorithms, and metaheuristics.

---

## 🎯 Learning Objectives

The course contributes to the achievement of the following Learning Outcomes (LO):

- **LO1**: Models complex engineering problems using heuristic and metaheuristic algorithms for the optimization of solutions.
- **LO2**: Solves complex engineering problems applying advanced algorithmic strategies.
- **LO3**: Communicates the solution to engineering problems through an oral and written proposal, applying the algorithmic strategies developed in the course and validating their efficiency through the experimentation of their results.



## 📚 Topics Covered

The course is organized into five chapters. Each chapter has a dedicated folder containing in-class exercises.

- **Chapter 1: Backtracking**: [`cap-01-backtracking/`](./cap-01-backtracking/)
- **Chapter 2: Dynamic Programming**: [`cap-02-dynamic-programming/`](./cap-02-dynamic-programming/)
- **Chapter 3: Heuristics - Greedy Algorithms**: [`cap-03-heuristics/`](./cap-03-heuristics/)
- **Chapter 4: Metaheuristics - GRASP Algorithm**: [`cap-04-metaheuristics/`](./cap-04-metaheuristics/)
- **Chapter 5: Evolutionary Algorithms**: [`cap-05-evolutionary-algorithms/`](./cap-05-evolutionary-algorithms/)

## 🛠️ Prerequisites

Students are expected to have prior knowledge of:

- **Data Structures**: lists, trees, graphs, hash tables, heaps.
- **Algorithms Fundamentals**: sorting, searching, basic graph traversal.
- **Mathematics**: discrete mathematics, probability, linear algebra basics.
- **Programming**: Proficiency in C++.

## ⚙️ Getting Started

### Prerequisites

Ensure you have the following tools installed:

- **Git** ≥ 2.30
- **C++ Compiler** (e.g., GCC/g++ ≥ 11, Clang)
- **CLion** (the official IDE we will use for this course)

### Clone the Repository

```bash
git clone https://github.com/jlcorcuera/pucp-adv-alg-2026-01.git
cd pucp-adv-alg-2026-01
```

### Running Code Examples

Each lecture/lab folder will contain specific setup and execution instructions if needed. In general, to compile and run the exercises using g++:

```bash
g++ -O2 -o solution solution.cpp
./solution
```
